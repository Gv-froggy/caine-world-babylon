# Caine World — Babylon.js Edition

A new iteration of the Caine World experiment.

This version rebuilds the project from the ground up using Babylon.js in order to support:

* procedural locomotion,
* inverse kinematics,
* physics-based interaction,
* autonomous AI behaviors,
* persistent memory systems,
* experimental digital creature simulation.

The goal is no longer just rendering a character, but creating a believable autonomous digital presence evolving inside a 3D world.

Status: early reconstruction phase.
